# Cyto Dollars React/Vite Site

Project structure:

```txt
public/
  cyto-logo.png
  cyto-logo-large.png
src/
  App.jsx
  main.jsx
index.html
package.json
vite.config.js
```

Run locally:

```bash
npm install
npm run dev
```

Then open the local URL shown in Terminal, usually:

```txt
http://localhost:5173/
```

Logo files are stored in `public/` and are referenced as `/cyto-logo.png` and `/cyto-logo-large.png`.
